REQUISITOS:
EN WINDOWS 7: DEBEMOS TENER INSTALADO EL SERVICE PACK 1 Y "Visual C++ 2012 Redistributable",
	      PRIMERO SE INSTALA SP1, PARA PODER INSTALAR "Visual C++ 2012 Redistributable".

EN WINDOWS 8, 8.1 Y 10: INSTALAR "Visual C++ 2012 Redistributable".
-----------------------------------------------------------------------------------------------------
PASO 1: EXTRAER EL ZIP SEGUN EL SISTEMA OPERATIVO.
	
	PARA APACHE:
	PARA 32BITS "php-5.6.30-Win32-VC11-x86 Thread Safe APACHE"
	PARA 64BITS "php-5.6.30-Win32-VC11-x64 Thread Safe APACHE"
	

	PARA IIS:
	PARA 32BITS "php-5.6.30-nts-Win32-VC11-x86 Non Thread Safe IIS"
	PARA 64BITS "php-5.6.30-nts-Win32-VC11-x64 Non Thread Safe IIS"
	
	LA RENOMBRAMOS A "php"

PASO 2: PEGAR LA CARPETA "php" EN EL DISCO C

PASO 3: ENTRAMOS A LA CARPETA DE PHP; CREAMOS UNA COPIA DE EL ARCHIVO "php.ini-development" Y
	LO RENOMBRAMOS A "php.ini"
	LO ABRIMOS PARA EDITARLO.

PASO 4: VAMOS A VERIFICAR LA RUTA DE EXTENSIONES , BUSCAMOS -> extension_dir = "ext",
	Y COMPROBAMOS QUE ESTE LA RUTA CORRECTA.
	DEBE QUEDAR ASI: extension_dir = "C:\php\ext"

	NOTA: SI TIENE PUNTO Y COMA(;) ADELANTE, QUITARLO;
	TAMBIEN SE USA PARA HABILITAR LAS EXTENSIONES

PASO 5: HABILITAR LAS SIGUIENTES EXTENSIONES:
	
	extension=php_curl.dll
	extension=php_openssl.dll
	extension=php_fileinfo.dll
	extension=php_gd2.dll
	extension=php_mysql.dll
	extension=php_mysqli.dll
	extension=php_openssl.dll
	extension=php_mbstring.dll	

PASO 6: POR ULTIMO AGREGAMOS LA VARIABLE DE ENTORNO DE PHP
	NOS DIRIGIMOS A:
	
	INICIO->CLICK DERECHO EN EQUIPO, PROPIEDADES->CONFIGURACION AVANZADA->
	OPCIONES AVANZADAS->VARIABLES DE ENTORNO->EN VARIABLES DE ENTORNO DEL SISTEMA SE BUSCA
	Path, SELECCIONAMOS Y LE DAMOS EDITAR->COMPROBAMOS EN VALOR DE VARIABLE QUE
	ESTE AGREGADO PHP(;C:\php), SINO ESTA, AGREGARLO, SOLO AGREGAR AL FINAL: ";C:\php"(SIN COMILLAS)

PASO 7: GUARDAMOS EL ARCHIVO.
